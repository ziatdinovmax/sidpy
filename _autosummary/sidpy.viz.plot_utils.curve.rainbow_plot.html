sidpy.viz.plot_utils.curve.rainbow_plot¶

sidpy.viz.plot_utils.curve.rainbow_plot(axis, x_vec, y_vec, num_steps=32, **kwargs)[source]¶

Plots the input against the output vector such that the color of the curve changes as a function of index

Parameters

  • axis (matplotlib.axes.Axes object) – Axis to plot the curve

  • x_vec (1D float numpy array) – vector that forms the X axis

  • y_vec (1D float numpy array) – vector that forms the Y axis

  • num_steps (unsigned int (Optional)) – Number of discrete color steps
