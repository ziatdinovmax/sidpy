sidpy¶

Python utilities for storing, visualizing, and processing Spectroscopic and Imaging Data (SID)

This utilities package supports other packages such as:

  • Data formatting:

    • pyNSID

    • pyUSID

  • Scientific analysis:

    • pycroscopy

    • ScopeReaders

    • BGlib
